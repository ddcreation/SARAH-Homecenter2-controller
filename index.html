Le plugin HC2 permet d'envoyer ou d'interroger votre box domotique Fibaro Home Center 2.

Exemples :

  • Allumer ou éteindre une lumiere: " SARAH, allume xxx"
  • Activer une scene: SARAH, "passe en mode veille" ou "SARAH, lance la scene xxx
  • demander le status d'un peripherique: "SARAH, quelle est la tempétarure du salon" ou "donne moi le statut de xxx"

 

1. Fonctionnement global

1.1 Controle grace au nom du periphérique:

Vous verrez ci-dessous qu'il suffit de renseigner le nom du module du home center 2 avec exactement la meme orthographe dans le fichier xml pour que le plugin controle votre module. Plus besoin de connaitre l'ID du peripherique. :)

  • le nom du module = lumiere salon.

  • out.action.dicatation = lumiere salon.

 

1.2 Detection automatique du type de périphérique:

Plus besoin non plus de connaitre le type du périphérique

  • Binary_Light
  • Dimmable_Light
  • Binary_Light
  • Humidity_sensor
  • Temperature_sensor
  • multi_level_sensor

Le plugin s'adapte au type de périphérique afin de mieux maitriser leurs fonctions.
